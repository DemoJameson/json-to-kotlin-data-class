在 Kotlin 中数据类（Data Class）能帮我们减少许多样板代码，自动生成 `equals`/`hashCode`/`toString` 等很多必要的方法。与后端通讯时流行的 JSON 数据十分适合用数据类来存储，如果有个工具能自动将 JSON 转化为数据类就好了。所以退化到只会用原生 Javascript DOM 的我就写了这个工具，没有花太多精力只做了点微小的工作（+1s）。
